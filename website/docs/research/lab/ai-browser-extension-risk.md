# Lab: AI Browser Extension Security Risk

## First seen
2026-04-11 · Ra brief

## What it is
AI-powered browser extensions inherit the user's logged-in session for every site they touch, creating a cross-site credential attack surface

## Why it's interesting
Single biggest new enterprise security risk in 2026 — extensions legitimately need page access, and users legitimately install them

## When we'd reach for it
Any time we consider building a Hyo browser extension — scope permissions narrowly from day one, never inherit global session tokens
