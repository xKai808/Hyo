# Lab: Hybrid Cloud-Trigger / Local-Runtime Agents

## First seen
2026-04-11 · Ra brief

## What it is
Small local program listens for triggers from the cloud and does the actual work on the user's own machine — not cloud-sandbox, not pure-local

## Why it's interesting
Cloud sandboxes can't reach the internet or touch personal files reliably; pure-local dies when the lid closes. Hybrid gets both.

## When we'd reach for it
Any agent that needs scheduled triggers plus real file/network access — Ra itself is the motivating example
