# Lab: Operating-Layer Controls for Onchain LLM Agents

## First seen
2026-04-30 · Ra brief

## What it is
Reliability framework for LLM agents managing real capital — structured controls, natural-language strategies, validated action gates — studied over 21 days of live deployment

## Why it's interesting
7.5M agent invocations trading real ETH; first serious empirical study of LLM agents under real financial pressure, not simulation

## When we'd reach for it
Building any agent with real-world consequences — payments, trading, infrastructure changes, resource allocation — any system where a bad call costs money
