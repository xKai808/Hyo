# Lab: OMEGA — AI That Invents ML Algorithms

## First seen
2026-04-30 · Ra brief

## What it is
End-to-end framework that takes a problem, generates novel ML algorithm ideas, writes executable code, evaluates it, and iterates — producing classifiers that outperform scikit-learn baselines

## Why it's interesting
Separates idea generation from code generation to reduce reward hacking; closes the loop between hypothesis and working code without a human in the middle

## When we'd reach for it
Early-stage exploration when you don't know which algorithm class fits your data, or when baseline comparison is eating disproportionate research time
