# Lab: Step-Level Optimization for Computer-Use Agents

## First seen
2026-04-30 · Ra brief

## What it is
A routing framework for GUI automation agents that sends trivial steps to lightweight models and reserves the large model for hard decisions

## Why it's interesting
Current computer-use agents invoke a large model at every click, including obvious ones — this cuts cost substantially without meaningful accuracy loss

## When we'd reach for it
Browser automation, desktop robotic process automation, form-filling, workflow testing at scale
